###########################
##  Structure du projet  ##
###########################

./src : dossier avec tous le codes nécéssaire pour faire run le dahsboard
 |
 |_ ./components: dossier avec le codes qui génère les composantes à introduire dans la layout (./pages/layout.py)
 |_ ./pages: dossier avec le code qui génère la page web du dahsboard divisé en tabs
 |_ ./utils: dossier avec toutes les fonctions nécéssaires à calculer les metrics du dashbaord
 |_ ./application.py: fichier à run pour génèrer toute l'application.
